get /restaurants
Get restaurants (restaurantsGet)
Returns the list of restaurants.

Query parameters

city (optional)
Query Parameter — The city code which will filter the restaurants.
slug (optional)
Query Parameter — The specific restaurant slug.
deal (optional)
Query Parameter — The type of flash deals. Default value is Earlybird.
allList (optional)
Query Parameter — Choose either to return related data to a restaurant such as seats, menus, timelist, turntimes, deadlines. Default value is true.
isShowcase (optional)
Query Parameter — The showcase flag. Default value is false.

Return type

array[Restaurant]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An array of restaurants

400

Invalid ID supplied

404

Restaurant not found
Up
